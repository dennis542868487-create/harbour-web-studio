# Delivery Checklist

## Before deployment
- Replace business name
- Replace phone, address, hours
- Replace domain in `business.siteUrl`
- Replace logo and favicon
- Replace shop photos
- Review homepage copy
- Review service page copy
- Review FAQ answers
- Review guide pages

## SEO checks
- Confirm canonical URLs
- Confirm sitemap domain
- Confirm robots sitemap URL
- Confirm Open Graph text
- Add Search Console verification file or meta tag
- Submit sitemap after launch

## Deployment checks
- Push repo to GitHub
- Import to Vercel
- Set production domain
- Update DNS records
- Confirm HTTPS
- Test homepage, FAQ, location, and service pages

## Post-launch checks
- Open live sitemap
- Open live robots.txt
- Test phone links
- Test map links
- Test mobile header/nav
- Request indexing for homepage and top service page
